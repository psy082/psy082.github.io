CSS Containing Block

м»Ён…ҢмқҙлӢқ л°•мҠӨлһҖ?

мӣ”л“ңмҷҖмқҙл“ңмӣ№ н‘ңмӨҖмқ„ л§Ңл“ңлҠ” лӢЁмІҙ w3cм—җм„ң мһ‘м„ұлҗң css кіөмӢқ л¬ём„ңм—җм„ң м»Ён…ҢмқҙлӢқ лё”лЎқмқ„ м–ҙл–»кІҢ м„ӨлӘ…н•ҳлҠ”м§Җ нҷ•мқён•ҙліҙм•ҳлӢӨ.

In CSS 2, many box positions and sizes are calculated with respect to the edges of a rectangular box called a containing block. In general, generated boxes act as containing blocks for descendant boxes; we say that a box вҖңestablishesвҖқ the containing block for its descendants. The phrase вҖңa boxвҖҷs containing blockвҖқ means вҖңthe containing block in which the box lives,вҖқ not the one it generates. - 9.1.2 Containing blocks. CSS 2.1 spec

кіөмӢқ л¬ём„ңм—җм„ңлҠ” м»Ён…ҢмқҙлӢқ лё”лЎқмқҙлқјлҠ” к°ңл…җмқ„ мҷң мӮ¬мҡ©н•ҳлҠ”м§ҖмҷҖ м»Ён…ҢмқҙлӢқ лё”лЎқмқҙ м–ҙл–»кІҢ м„Өм •лҗҳлҠ”м§ҖлҘј к°„лһөн•ҳкІҢ м„ӨлӘ…н•ңлӢӨ.(мһҗм„ён•ң м„ӨлӘ…мқҖ мҠӨнҺҷмқҳ лӢӨлҘё мұ•н„°м—җм„ң м„ӨлӘ…мқ„ н•ҳкі  мһҲлӢӨ. нӣ„мҲ н•  мҳҲм •) м„ӨлӘ…м—җ л”°лҘҙл©ҙ мҡ”мҶҢл“Ө(boxл“Ө)мқҳ position(CSSм—җм„ңмқҳ positionмқҖ лӢЁмҲңнһҲ мң„м№ҳк°Җ м•„лӢҲкё° л•Ңл¬ём—җ)кіј нҒ¬кё°(лҶ’мқҙмҷҖ л„Ҳл№„)лҘј кі„мӮ°н•ҳкё° мң„н•ң кё°мӨҖм җмқ„ м •н•ҳкё° мң„н•ҙ н•„мҡ”н•ң к°ңл…җмқҙлӢӨ. мүҪкІҢ л§җн•ҳл©ҙ к°Ғк°Ғмқҳ htmlмҡ”мҶҢ(box)к°Җ мһҲлҠ”(lives) кіімқҙ м»Ён…ҢмқҙлӢқ лё”лЎқмқҙлӢӨ.

м»Ён…ҢмқҙлӢқ лё”лЎқ м„Өм •мқҳ мһҗм„ён•ң лӮҙмҡ©мқ„ м•Ңл ӨмЈјлҠ” мһҲлҠ” лӢӨлҘё мұ•н„°лҘј нҷ•мқён•ҙ ліҙм•ҳлӢӨ. н•ҙлӢ№ лӮҙмҡ©мқҖ кёёкё° л•Ңл¬ём—җ мқёмҡ©н•ҳм§ҖлҠ” м•Ҡм§Җл§Ң м—¬кё°м—җм„ң нҷ•мқён•  мҲҳ мһҲлӢӨ.

м»Ён…ҢмқҙлӢқ л°•мҠӨ мһҗм„ёнһҲ мӮҙнҺҙліҙкё°

мҡ°м„  м»Ён…ҢмқҙлӢқ л°•мҠӨм—җ лҢҖн•ҙм„ң мҳӨн•ҙн•ҳкё° мү¬мҡҙ л¶Җл¶„мқ„ мқҙм•јкё° н•ҳкі  м»Ён…ҢмқҙлӢқ л°•мҠӨк°Җ positionк°’м—җ л”°лқјм„ң м–ҙл–»кІҢ м •мқҳлҗҳлҠ”м§Җ мӮҙнҺҙліҙл Өкі  н•ңлӢӨ.

htmlмқ„ мһ‘м„ұн•  л•Ң, мҡ”мҶҢ м•Ҳм—җ мҡ”мҶҢк°Җ мһҲлҠ” кІҪмҡ° мҡ”мҶҢлҘј н’ҲмқҖ мҡ”мҶҢлҘј л¶ҖлӘЁ(мғҒмң„) мҡ”мҶҢлқјкі  н•ҳкі , мҡ”мҶҢ м•Ҳм—җ мһҲлҠ” мҡ”мҶҢлҘј мһҗмӢқ(н•ҳмң„) мҡ”мҶҢлқјкі  н•ңлӢӨ. html л¬ём„ңл“ӨмқҖ мқҙл ҮкІҢ мҡ”мҶҢл“Ө к°„мқҳ кі„мёөл“Өмқ„ нҳ•м„ұн•ҳл©ҙм„ң мһ‘м„ұлҗңлӢӨ. мҡ°лҰ¬лҠ” лёҢлқјмҡ°м Җк°Җ мқҙ html л¬ём„ңлҘј л ҢлҚ”л§Ғн•ҳкі  к·ёл ӨлӮё нҷ”л©ҙмқ„ ліҙл©ҙм„ң м»Ён…ҢмқҙлӢқ лё”лЎқмқҖ л¶ҖлӘЁмҡ”мҶҢмқј кІғмқҙлқјкі  м¶”мёЎн•ҳкІҢ лҗңлӢӨ. мҠӨнҺҷмқҳ м„ӨлӘ…м—җ л”°лҘҙл©ҙ лҢҖл¶Җл¶„мқҳ кІҪмҡ° мқҙ м¶”мёЎмқҖ л§һлӢӨ. н•ҳм§Җл§Ң н•ӯмғҒ к·ёлҹ° кІғмқҖ м•„лӢҲлӢӨ. мҠӨнҺҷмқҖ л¶ҖлӘЁмҡ”мҶҢк°Җ м»Ён…ҢмқҙлӢқ лё”лЎқмқҙ м•„лӢҢ кІҪмҡ°м—җ лҢҖн•ң лӮҙмҡ©, м»Ён…ҢмқҙлӢқ лё”лЎқм—җ лҢҖн•ң мһҗм„ён•ң м •мқҳлҘј м„ӨлӘ…н•ңлӢӨ.

  1. root element(л¶ҖлӘЁ мҡ”мҶҢк°Җ м—ҶлҠ” мң мқјн•ң мҡ”мҶҢ)к°Җ мһҲлҠ” м§ҒмӮ¬к°Ғнҳ•мқҳ мҳҒм—ӯмқҖ мөңмҙҲ м»Ён…ҢмқҙлӢқ лё”лЎқ(initial containing block)мқҙлӢӨ.
